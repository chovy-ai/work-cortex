# 配方 · 数据表格（明细 / 对比表 → 图片）

method: code —— 渲染成图片，不要让栅格模型画表格（数字会被编造、对不齐）。

## 工具选择（按可得性降级）
1. **matplotlib** `ax.table`（或 `df.plot(table=True)`）`savefig` 导 PNG —— 最稳，无需浏览器；
2. 或写 **HTML 表格**用 headless 浏览器截图（有则用）；
3. 或手写 **SVG** 表格。
**不要联网装依赖**；把脚本与成品图一起存产物目录。

## 规范
- 表头加粗 / 底色区分；
- 数字**右对齐**、千分位分隔、统一小数位；文本左对齐；
- 合计 / 关键行高亮（底色或加粗）；
- 列数控制在一屏内（>8 列考虑转置或拆表）；行多时只渲染 Top-N 并在标题注明；
- 细边框或斑马纹择一，别又粗边框又斑马；
- 中文表用中文表头（matplotlib 需设中文字体，缺则退英文表头避免豆腐块）。

## 输出
- 导 PNG 存产物目录（如 `metrics-table-1.png`）；脚本一并存；
- `images[].path` 指向 PNG；`revised_prompt` 写明渲染方式与是否截断（如「matplotlib table，Top 20 行」）。
