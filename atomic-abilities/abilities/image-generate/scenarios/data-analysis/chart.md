# 配方 · 数据分析图（折线 / 柱状 / 饼 / 散点 / 漏斗 / 分布）

method: code —— 写代码渲染，禁止用栅格图像模型编造数据图。

## 工具选择（按可得性降级）
1. 首选 **matplotlib**（`savefig` 导 PNG，`dpi>=150`）；
2. 或 **plotly**（`write_image`，需 kaleido）；
3. 都没有则用 **ECharts/Vega HTML** 截图，或手写 SVG。
**不要联网装依赖**；缺库就降级。把脚本（`.py`）与成品图一起存产物目录。

## 选图型（按数据意图）
- 趋势 / 时间序列 → 折线图；
- 类别对比 → 柱状图（横条更适合长标签 / 多类别）；
- 构成占比 → 饼 / 环形 / 堆叠柱（类别 >6 用堆叠柱，别用饼）；
- 同环比 → 分组柱 或 双轴折线；
- 分布 → 直方图 / 箱线图；
- 相关 → 散点；
- 转化 → 漏斗（逐级条形，标转化率）。

## 规范
- 必有标题；坐标轴标签**带单位**；图例在不挡数据处；
- 柱状按值排序（除非类别本身有序，如时间 / 等级）；
- 数据点少时直接标数值；千分位分隔；
- 配色用色盲友好palette（如 tab10 / Okabe-Ito），≤6 色；
- 去除多余网格 / 边框（chartjunk）；中文图用中文标题与标签（matplotlib 需设中文字体，缺则用英文标签别出豆腐块）。

## 输出
- 导 PNG 存产物目录，文件名表意（如 `dau-trend-1.png`）；脚本一并存；
- `images[].path` 指向 PNG；`revised_prompt` 写明图型与工具（如「matplotlib 折线，按周聚合」）。
